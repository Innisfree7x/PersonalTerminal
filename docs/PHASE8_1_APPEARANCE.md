# PHASE 8.1 — Appearance Upgrade (Premium Visual Pass)

## Ziel
Die Marketing-Site von solide auf premium bringen, ohne die Conversion zu verschlechtern.

## Finaler Scope (abgeschlossen)
- Dark-first Brand-Look auf Schwarz/Rot/Gold konsolidiert.
- Hero-Hierarchie geschärft (klarere CTA-Reihenfolge, bessere Lesbarkeit).
- Navbar visuell beruhigt (Spacing und Balance verbessert).
- Branding auf `INNIS` umgestellt inklusive neuer Logo/Lockup-Komponenten.
- Premium-Klassen für wiederverwendbare UI-Flächen eingeführt.

## Was live ist
- Premium-Style-Utilities:
  - `premium-kicker`
  - `premium-chip`
  - `premium-cta-primary`
  - `premium-cta-secondary`
- Verfeinerte Oberflächen:
  - stärkere Karten-Tiefe
  - konsistentere Border/Glow-Intensitäten
  - bessere Abstände in Hero/Navigationsbereich
- Hero-Polish:
  - headline scale reduziert und ausbalanciert
  - CTA-Paar visuell sauber getrennt (`Get Started` dominiert)

## Was bewusst verworfen wurde
- Dynamischer Hero-Spotlight-Effekt wurde entfernt (Flicker/Instabilität).
- Companion-Hero-Card wurde entfernt (visuelles Rauschen, unklare Story im First View).

## QA-Status (8.1 Final)
- `npm run type-check` ✅
- `npm run lint` ✅
- `npm run build` ✅

## Ergebnis
Phase 8.1 ist abgeschlossen. Die Seite ist visuell hochwertiger, gleichzeitig stabiler und conversion-fokussierter.
