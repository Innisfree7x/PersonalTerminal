## Risk Assessment

### Remaining external dependency risks
- If Google Cloud Console does not whitelist the exact callback URI for the deployed domain, OAuth will still fail.

### Operational requirement
- In Google Cloud Console OAuth client, authorized redirect must include exactly:
  - `https://<your-domain>/api/auth/google/callback`

## Handoff Notes for Future Agents
1. Do not split redirect-uri logic across routes again; use `lib/google/oauth.ts` as the single source.
2. Do not reintroduce dual-control cmdk state in `CommandPalette`.
3. Keep Focus visual preference key stable unless a migration is planned.
4. Any visual extension on Focus must preserve quote vertical centering and readability contrast.
